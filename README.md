**FFmpegBuilds** 是自动编译windows,linux,macos平台的ffmpeg,每月5号将会自动执行

## 说明
- 官方的ffmpeg并不支持x86平台且官方的ffmpeg是全功能版本 但是很多时候并需要这么多的功能 所以此项目只做了基础功能的编译包含编码转码等一些基础功能所以编译后的程序大小是官方程序大小的1/3
